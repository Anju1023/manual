🌅毎日の作業手順

⏰ 作業開始前のチェック

毎日の作業を始める前に、必ずバックアップを取り、前日の作業が正しく完了していることを確認してください。

📋 作業1：天板並べ個数表の作成

🎯 目的

今日製造する商品を天板にどのように並べるかの指示書を作成します。

📍 使用シート

昼間用シート（24時間体制の場合は夜間用シートも）

🔧 操作手順

  1. ファイルを開く
    • CB商品データ.xlsmを開く
    • マクロの有効化を確認（「コンテンツの有効化」をクリック）
  2. 操作シートに移動
    • 画面下部の「操作」タブをクリック
    • ボタン一覧が表示されることを確認
  3. 更新ボタンを実行
    • 「昼間用更新」ボタンを1回だけクリック
    • 処理完了まで待機（他の操作は行わない）
    • 処理時間：約30秒〜1分
  4. 結果確認
    • 「昼間用」シートに移動
    • 今日の日付が正しく表示されているか確認
    • 商品名、個数、並べ方が表示されているか確認
  5. 印刷
    • Ctrl+P で印刷画面を開く
    • 印刷範囲が正しく設定されていることを確認
    • 必要部数を印刷

⚠️ 注意事項

  • ボタンは必ず1回だけクリックしてください
  • 処理中にExcelを操作したり、他のアプリケーションを開いたりしないでください
  • エラーが表示された場合は、処理を中断して上席者に報告してください

📄 作業2：明細書の作成

🎯 目的

各店舗への納品内容と金額を記載した明細書を作成します。

📍 使用シート

伝票用シート（自社店舗用）、クリエ伝票用シート（他社店舗用）

🔧 操作手順

  1. 伝票更新の実行
    • 「操作」シートで「伝票用更新」ボタンをクリック
    • 続いて「クリエ伝票用更新」ボタンをクリック
    • 各処理の完了を待つ
  2. 金額確認
    • 「伝票用」シートで各店舗の合計金額を確認
    • 明らかに異常な金額がないかチェック
    • 前日の金額と大幅に異なる場合は確認
  3. 印刷実行
    • 各シートを印刷（店舗数に応じて複数ページ）
    • 印刷後、数値に間違いがないか最終確認

💾 作業3：納品記録PDFの作成

🎯 目的

各店舗への納品実績を記録として保存します。後日の問い合わせ対応や実績管理に使用します。

🔧 操作手順

  1. PDF出力準備
    • 各店舗シートで納品データが最新であることを確認
    • 「操作」シートで「PDF出力」ボタンをクリック
  2. 保存場所の指定
    • 保存ダイアログで適切なフォルダを選択
    • ファイル名に日付を含める（例：納品記録_20240722.pdf）
  3. 内容確認
    • 作成されたPDFを開いて内容を確認
    • 全店舗のデータが含まれているかチェック

📅週次作業

📅 実施タイミング

毎週日曜日の夕方に実施することを推奨します。翌週の準備を整え、スムーズな業務開始につなげます。

🔄 作業1：全シート更新作業

🎯 目的

一週間の業務データを整理し、各シートの計算結果を最新状態に更新します。

🔧 操作手順

  1. 更新前のバックアップ
    • CB商品データ.xlsmのコピーを作成
    • ファイル名に日付を追加して保存
  2. 製造数更新
    • 「操作」シートで「製造数更新」ボタンをクリック
    • 処理完了まで約2-3分待機
  3. 生地出し更新
    • 「冷凍生地出し更新」ボタンをクリック
    • 「スクラッチ生地出し更新」ボタンをクリック
  4. ラベル出し更新
    • 「ラベル出し更新」ボタンをクリック
    • 「クリエラベル出し更新」ボタンをクリック

🖨️ 作業2：厨房用表の印刷

📋 印刷対象シート

  • 冷凍生地出し - 冷凍生地の作業計画表
  • スクラッチ生地出し - 生地作りの作業計画表
  • 並べる表 - 天板配置の計画表
  • ラベル出し - 商品ラベル作成用

🔧 印刷手順

  1. 各シートを順番に選択
  2. 印刷プレビューで内容と範囲を確認
  3. 必要部数を印刷（通常は2部）
  4. 印刷物を厨房の指定場所に掲示

📦 作業3：発注表作成

🎯 目的

翌週に必要な材料を自動計算し、仕入先別の発注表を作成します。

🔧 操作手順

  1. 発注データ更新
    • 「操作」シートで「発注表作成」ボタンをクリック
    • レシピデータと製造数から必要量を自動計算
  2. 発注量の確認
    • 「発注」シートで計算結果を確認
    • 異常に多い/少ない発注量がないかチェック
    • 在庫状況と照らし合わせて調整
  3. 仕入先別発注表の出力
    • 「仕入先別発注表作成」ボタンをクリック
    • 各仕入先用の発注表を印刷
    • FAXまたはメールで各仕入先に送信

🧾 作業4：消耗品発注

📋 確認項目

  • 包装資材（袋、容器、ラベル用紙）
  • 清掃用品（洗剤、タオル、ブラシ）
  • 事務用品（印刷用紙、インク、文具）
  • 安全用品（手袋、マスク、エプロン）

🔧 発注手順

  1. 各消耗品の在庫状況を目視確認
  2. 「消耗品管理表」（Excelの別ファイル）を更新
  3. 必要な商品を発注
  4. 発注記録を管理表に記載

📆月次作業

📅 実施タイミング

毎月最終営業日に実施します。翌月の準備と当月の実績整理を行います。

👥 作業1：シフト表作成

🎯 目的

翌月の従業員シフトを作成し、適切な人員配置を計画します。

🔧 操作手順

  1. シフト希望の収集
    • 従業員からの休暇希望を確認
    • 繁忙期・閑散期を考慮
  2. シフト表の作成
    • 専用のシフト管理Excelファイルを使用
    • 製造量に応じた適切な人員配置
    • 労働時間の法的制限を遵守
  3. 確認と配布
    • 作成したシフト表を印刷
    • 従業員への配布・掲示

📊 作業2：棚卸表の本社報告

🎯 目的

月末時点での材料在庫を正確に把握し、本社に報告します。

🔧 操作手順

  1. 在庫実地確認
    • 冷凍庫・冷蔵庫・常温保管の全材料を確認
    • 数量と消費期限を記録
  2. 棚卸表への入力
    • 専用の棚卸表Excelファイルに入力
    • 理論在庫との差異を確認
    • 差異がある場合は原因を調査
  3. 本社への報告
    • 完成した棚卸表をメール添付で送信
    • 特記事項がある場合は別途連絡

🆕 作業3：新商品登録

📋 登録情報

  • 商品ID（カテゴリ頭文字+4桁数字）
  • 商品名
  • 売価（一般・クリエ用）
  • 消費期限
  • 生地名・製造方法
  • 包装資材・ラベル番号

⚠️ 注意事項

新商品登録は必ず上席者の指示のもとで行ってください。間違った登録は全システムに影響します。

🔚 作業4：終売商品の処理

🎯 目的

販売終了となった商品をシステムから非表示にし、製造計画から除外します。

🔧 操作手順

  1. 終売商品の確認
    • 本社からの終売指示を確認
    • 在庫状況を最終チェック
  2. システムでの処理
    • 「商品データ」シートで該当商品を検索
    • 「終売」列に「○」を入力
    • 各種表示から自動的に除外される

⚠️ 重要な注意

商品データは削除せず、非表示処理してください。過去のデータ参照で必要になる場合があります。

✏️データ変更・修正作業

🚨 データ変更の重要ルール

データ変更は40店舗すべてに影響するため、必ず上席者の承認を得てから実行してください。

🏪 店舗の納品個数変更

📞 変更依頼への対応

店舗から納品個数の変更依頼があった場合の対応手順です。

🔧 操作手順

  1. 変更内容の確認
    • 依頼店舗名・商品名・変更後個数を正確に記録
    • 変更理由を確認（イベント、天候等）
    • 変更日時を明確にする
  2. 上席者への報告
    • 変更内容を上席者に報告
    • 承認を得てから作業開始
  3. システムでの変更
    • 該当店舗のシートを開く
    • 対象日・対象商品のセルを見つける
    • 新しい個数を入力
  4. 影響範囲の確認
    • 「製造数」シートで合計製造数の変化を確認
    • 「生地出し」シートで必要生地量の変化を確認
    • 「発注」シートで材料必要量の変化を確認
  5. 関連書類の更新
    • 変更後の明細書を再作成
    • 製造指示書を更新
    • 変更記録を作業日報に記載

💰 価格変更への対応

🎯 価格改定時の作業

本社からの価格改定指示に基づく売価変更の手順です。

⚠️ 事前準備

  • 価格改定指示書の内容を十分に確認
  • 変更対象商品のリストを作成
  • 実施日時を確認
  • 必ずバックアップを作成

🔧 変更手順

  1. 「商品データ」シートを開く
  2. 対象商品を検索（Ctrl+Fで検索）
  3. 「売価」列または「クリエ売価」列を更新
  4. 変更後の価格が正しく入力されているか確認
  5. 「価格改定後」シートで変更内容を記録

📝 変更作業の記録

📋 記録すべき内容

  • 変更日時
  • 変更者名
  • 変更内容（変更前→変更後）
  • 変更理由
  • 承認者名
  • 影響範囲

💾 記録方法

専用の変更記録簿（Excelファイル）に記載し、定期的にバックアップを取ってください。

🚫 してはいけないこと

  • 口頭依頼のみでの変更（必ず書面またはメールで確認）
  • 複数の変更を同時に実行
  • 変更後の確認を怠る
  • バックアップなしでの重要な変更
  • 承認なしでの独断変更
